# Machine Learning Foundations

Trying to cover core machine learning concepts and model-building workflows.

## Project Overview

The goal of this repository is to build practical intuition for machine learning by working through a sequence of focused notebooks. Each notebook highlights a specific concept or model and helps reinforce the basics of training, comparing, and improving machine learning systems.

## Topics Covered

This repository includes notebooks on:

- Linear regression
- Nonlinear regression
- Polynomial vs. linear models
- K-nearest neighbors (KNN) classification
- Perceptron models
- Pipelines and hyperparameter tuning

## Notebook Guide

- `01_linear_regression.ipynb`  
  Introduces linear regression and basic predictive modeling.

- `02_nonlinear_regression.ipynb`  
  Explores nonlinear regression and how more flexible relationships can be modeled.

- `03_polynomial_vs_linear.ipynb`  
  Compares polynomial and linear models to show differences in fit and complexity.

- `04_knn_classification.ipynb`  
  Covers K-nearest neighbors for classification tasks.

- `05_perceptron.ipynb`  
  Introduces the perceptron as a foundational linear classifier.

- `06_pipelines_and_hyperparameters.ipynb`  
  Demonstrates how to organize machine learning workflows and tune model settings more effectively.

## Learning Goals

This project was built to strengthen understanding of:

- supervised learning basics
- regression vs. classification
- model comparison
- underfitting and overfitting
- feature/model complexity
- training workflows
- hyperparameter tuning
